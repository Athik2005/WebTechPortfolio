Farmers' App for Wool Monitoring
Short Summary:
The purpose of the Wool Monitoring app is to help farmers manage and keep an eye on the quantity, quality, and market value of wool produced on their properties. With the help of this app, you can keep tabs on wool production, document fiber properties, and guarantee the best possible wool grading and pricing. Farmers can improve the way they produce wool, keep track of their records, and decide whether to sell their wool on the open market by using this app. Modules of operation: User Verification: Sign up or log in: Safe access to the application featuring login and registration options for farmers. Profile of the User: Control the choices and details of farmers. Inventory Control for Wool: Wool Batch Entry: Enter the weight, fiber length, and micron count of each fresh batch of wool. Wool Grading: Based on industry norms, wool grades can be entered manually or automatically. Batch History: Access and handle previous information for every batch of wool. Observation and Evaluation: Monitoring and evaluating fiber properties including length, strength, and micron count is part of the quality analysis process. Get up-to-date wool market information to gain insight into pricing patterns.

Project 3


CNF Table: Short Description and Working Modules
Short Description:
 The CNF (Context-Free Grammar in Chomsky Normal Form) Table is a tool designed to assist in the analysis and parsing of context-free grammars by converting them into Chomsky Normal Form. CNF is a standardized form of grammar that simplifies the parsing process, making it easier to apply algorithms like the CYK (Cocke-Younger-Kasami) parsing algorithm. This tool is especially useful for students, educators, and developers working on compiler design, formal language theory, and related fields.
Working Modules:
1. Grammar Input Module:
- Users can input a context-free grammar (CFG) in a user-friendly format. - Supports multiple rules and productions.
2. Grammar Conversion Module:
- Converts the input CFG into Chomsky Normal Form (CNF). - Simplifies the grammar by removing null productions, unit productions, and unnecessary symbols.
3. Parsing Table Generation:
- Generates a CNF table that shows the transformation of each production rule. - Provides detailed steps of the conversion process for educational purposes.
4. Validation Module:
- Validates the input grammar to ensure it meets the requirements of a context-free grammar. - Checks for common errors and provides suggestions for corrections.
5. CYK Algorithm Integration:
- Integrates the CYK parsing algorithm to demonstrate the application of CNF in parsing strings. - Users can input strings to see how they are parsed using the CNF table.
6. Export & Reporting:
- Allows users to export the CNF table and related data in various formats (e.g., PDF, CSV). - Generates reports detailing the grammar conversion and parsing process.
7. User Interface & Visualization:
- Provides a clear and intuitive interface for users to interact with the CNF table. - Includes visual aids to help users understand the grammar conversion and parsing process. This tool serves as both an educational resource and a practical application for those working with context-free grammars and their transformations.
